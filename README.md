# astrbot_plugin_qqadmin22222

AstrBot QQ 群管理插件，仅支持 `aiocqhttp` 平台。

## 功能简介

本插件用于 QQ 群日常管理，提供常见群管能力和自动化风控能力。

## 主要功能

- 禁言 / 解禁
- 全体禁言开关
- 踢人 / 踢出并拉黑
- 修改群名片
- 设置群头衔
- 设置/取消管理员
- 撤回消息
- 设置群头像
- 修改群名
- 设置/取消精华消息
- 发布/查看群公告
- 投票禁言
- 自定义违禁词
- 内置违禁词检测
- 刷屏检测
- 宵禁
- 进群审核
- 进群黑白词
- 进群等级/次数限制
- 进群黑名单
- 进群欢迎
- 退群通知
- 退群拉黑
- 群成员活跃信息
- 清理群友
- 群文件上传/删除
- 前端配置页面
- 默认群模板与分群独立配置
- 权限分级控制

## 权限层级

```text
超级管理员 > 群主 > 管理员 > 成员
```

## 安装

将插件目录放入 AstrBot：

```text
data/plugins/astrbot_plugin_qqadmin
```

然后重载插件或重启 AstrBot。

## 使用

在群内发送：

```text
群管帮助
```

或：

```text
/群管帮助
```

查看完整命令列表。

## 常用命令示例

```text

解禁 @用户
开启全禁
关闭全禁
踢了 @用户
拉黑 @用户
改名 新昵称 @用户
头衔 头衔名 @用户
撤回
撤回 @用户 10
设置群名 新群名
发布群公告 内容
设置禁词 词1 词2
刷屏禁言 60
开启宵禁 23:00 07:00
关闭宵禁
进群审核 开
进群审核 关
群友信息
清理群友 30 1
```

## 配置说明

插件支持配置页面管理：

- 默认群配置
- 单群独立配置
- 权限配置
- 违禁词配置
- 进群审核配置
- 宵禁配置
- 通知配置

发布版本不会包含本地运行数据、数据库、群号、用户 ID 或 Token。
